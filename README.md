# Library Management System

A simple Library Management System built using Python and SQLite. This project demonstrates basic database operations and CRUD functionality with a menu-driven interface.

---

## 📌 Features

* Add new books
* View all books
* Search books by ID
* Update book quantity
* Delete books
* Add members
* View members
* Search members
* Delete members
* Issue books to members
* Return books
* Fine calculation for late returns
* Admin login system

---

## 🛠️ Technologies Used

* Python
* SQLite3 (Database)

## 🚀 How to Run

1. Install Python (3.x)
2. Download or clone the repository
3. Open terminal in project folder
4. Run the program:

```bash
python main.py
```
